Wasm Beyond the Browser
August 9th, 2019 @ UC Berkeley

Workshop Overview

WebAssembly is an emerging standard for sandboxing and running platform independent code that has seen rapid adoption. The goal of this mini workshop is to provide a forum for discussing different application domains that can reap the benefits of Wasmâ€™s portability, the challenges with using Wasm in these domains, and potential extensions to the language.

The format of the workshop will consist of a few short talks, breakout sessions, and a dinner. The main purpose of this workshop is to exchange ideas and seed collaborations around WebAssembly. We will use video conferencing software to allow for remote participation.

Intended Audience

WebAssembly beyond the Browser seeks to bring together researchers, core WebAssembly developers, and developers exploring the implications of using WebAssembly in the browser and beyond.

Topics of interest (including, but not limited to)

  • Wasm runtime systems and libraries
  • Applications of Wasm outside the browser (e.g., IoT, serverless, blockchain, etc.)
  • Wasm language extensions for security, performance, etc.
  • Support for non-C-like language
  • Host/Wasm interfaces and marshaling
